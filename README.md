# Phoenix AI

Projeto para construir uma operação de conteúdo com inteligência artificial, automação e análise de performance.

## Estrutura

- `docs/`: visão, roadmap, brandbook e manual de IA.
- `prompts/`: prompts operacionais para roteiro, legenda, hashtag e storyboard.
- `automation/`: fluxos, integrações e automações.
- `backend/`: serviços, APIs e lógica de servidor.
- `frontend/`: interfaces e experiências do usuário.
- `assets/`: imagens, marcas, vídeos e materiais visuais.
- `analytics/`: métricas, relatórios e experimentos.
