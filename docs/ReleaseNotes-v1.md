# Release Notes v1

## Phoenix AI 1.0.0-rc1

Esta release candidate consolida a Phoenix AI como plataforma empresarial de IA criativa.

## Inclui

- Runtime multiagente.
- Studio e API.
- OpenAPI e SDK TypeScript.
- Multi-tenant, RBAC, Identity, OIDC e Secrets.
- Workflow Builder, Scheduler e Event Bus.
- Knowledge Graph, Hybrid RAG, Memory e Learning.
- Strategy Engine, Evaluation Engine e Quality Pipeline.
- Asset, Publishing e Provider Architecture.
- Cost Intelligence, budgets, quotas e cache semantico.
- Docker, health checks, diagnostics, backup e restore.

## Antes da v1.0.0 final

- Executar testes reais com providers pagos.
- Validar carga basica com dados reais.
- Revisar documentacao operacional com usuarios externos.
- Congelar novas features durante estabilizacao.
